Package安裝

  • 在VSCode的最下面，有terminal，可以在裡面打 pip install ... 就好
  • 想要移除package，就打 pip uninstall ...
  • 正常來說，terminal安裝完，在VSCode的script頁面就可以讀到了。但有時候會讀不到，這時就得重啟 VSCode( cmd+shift+p → reload window )，他才讀得到新安裝的版本...(不知道有沒有更好的做法)
  • 以下整理FAQ:

Q: 公司的個人筆電，無法安裝package

  • 特別提醒，公司的個人電腦，用 pip 時，要多一個 --user 的參數，變成 pip install --user ... 才能順利安裝

Q: pd.read_excel 不能work

  • 之前安裝pandas後，還要安裝 xlrd 這個 package，才能讀 excel。
  • 但現在 xlrd 只支援讀 .xls 檔，不鳥 .xlsx 了。所以現在要安裝另一個 package才能用
  • 安裝 openpyxl 
    pip install pandas
pip install openpyxl
  • 讀檔時這樣讀
    import pandas as pd
df = pd.read_excel("my_file.xlsx", engine='openpyxl')
  • 這個解法搜尋自 google，原始網頁在這 [連結]
