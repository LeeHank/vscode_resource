Step 0: 開啟 sql server

  • 這取決於你要連到哪一個 sql server。
  • 如果在公司，那就是連到一個遠端server。那這個遠端server本來就已經開啟了(或者說，開啟與否，不是我們可以掌管的，會有DBA人員在維護，所以如果這個 server 沒打開，那要請他幫我們打開)
  • 如果是自己私人用，比如說，我在 local 開了個 sql server，那這時候就是先去打開這個 server

Step 1: 在 vscode 中下載 MySQL extension

  • 搜尋與下載這個 extension

Step 2: 設定連結

Step 3: 選取db

  • 連線成功後，會變成下面的畫面，列出所有的 databases
  • 點選想要進入的 db，然後按 open query 的 icon，就會進入此db。此時這個 db 會顯示 Active

Step 4: 開 sql 文件搞事囉

  • 如果這個 sql 文件，要執行的 table 在別的 db，那就回到 step 3，去進入要操作的 db，那再回來這份文件按 ctrl + enter，就可以成功執行了。
